# Intern Assignment: Git, Shell Scripting, Docker & Streamlit

This repository contains my internship assignment work covering Git fundamentals, Shell scripting,
Docker basics, and a Streamlit application containerized using Docker.

## Folder Overview

### 1. Git Assignments
Contains all Git-related tasks including:
- Repository initialization and commits
- Branching, merging, and remote operations
- Merge conflict handling
- Undoing changes and interactive rebasing

### 2. Shell Scripting
Contains shell scripts covering:
- File system navigation and automation
- Variables, conditionals, loops, and functions
- Text processing using grep, sed, and awk
- Error handling and debugging

### 3. Docker & Streamlit Project
Contains:
- Docker basics and container operations
- Dockerfile for Python and Streamlit application
- Streamlit app containerized using Docker
- Git-based workflow with branches and releases

## Purpose
This repository demonstrates hands-on understanding of:
- Version control using Git
- Linux shell scripting fundamentals
- Containerization using Docker
- Deploying applications using Streamlit + Docker
